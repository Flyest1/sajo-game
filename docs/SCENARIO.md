# 강호의 별 — 분기형 캠페인 시나리오 설계서 (v2 리부트)

> 본 문서는 AI(Claude)가 작성했습니다.
> 김용(金庸) 원작 4부작의 사건·인물을 뼈대로 하되, 게임 내 모든 대사·연출은 팬메이드 창작입니다.

---

## 1. 설계 목표

현행 19장은 "한 줄로 이어지는 요약 하이라이트"였다. v2 리부트의 목표는 세 가지다.

1. **원작 충실도**: 각 권의 핵심 사건 흐름(기연 → 갈등 → 오해/비극 → 결전)을 원작 순서대로 밟는다.
2. **복잡 분기**: 선택지·전투 결과·수집 플래그가 루트를 가르고, 이후 권까지 영향을 미친다.
3. **스테이지 다양성**: 전멸전 일변도에서 벗어나 호위전·잠입전·설득전·단독전·생존전·진영선택전을 도입한다.

## 2. 현행 19장 평가 (원작 대비)

| 항목 | 현행 | 문제 |
|---|---|---|
| 사조영웅전 | 7장 | 우가촌 전사(前史), 양강 5회 배신 서사, 강남오괴 참극→황약사 오해, 철장방/영고/일등대사 생략 |
| 신조협려 | 2장 | 절정곡·정화독·16년 이별·독고구패 검총·곽양 서사 전부 생략 |
| 의천도룡기 | 5장 | 빙화도/사손 전사 생략, 광명정이 "성곤 토벌"로 단순화(원작은 육대파 vs 명교), 주지약·영사도 부재 |
| 천룡팔부 | 5장 | 취현장 혈투 생략, 아주 서사(핵심 비극) 부재, 소원산·모용박 진상 생략 |
| 공통 | — | 분기 없음, 전투 목표 단조(전멸/보스/방어 3종), 사이 서사는 내레이션 1줄 |

## 3. 분기 시스템 설계

### 3.1 스테이지 그래프
챕터 배열 → **노드 그래프**로 전환. 노드 종류:

- `battle` 전투 스테이지 (승리 조건 + 서브 목표)
- `talk` 대화 전용 스테이지 (전투 없음, 선택지 포함 가능)
- `choice` 루트 선택 노드 (지도에서 다음 행선지 선택)
- `gaiden` 외전 스테이지 (해금 조건 충족 시 루트 맵에 표시)

`next`는 단일 id, 조건 분기 `[{if, to}]`, 선택 분기 `{options}` 세 형태를 지원.

### 3.2 플래그 시스템
`G.flags = {}` — 대화 선택지, 전투 서브 목표, 영입/생존 결과가 기록되어 이후 분기·대사·엔딩을 결정. 세이브 v2에 포함.

### 3.3 신규 전투 룰
| 룰 | 설명 | 사용 예 |
|---|---|---|
| 서브 목표 | "N턴 내 승리", "특정 적을 특정 아군으로 격파", "가옥 파괴 0" 등 → 플래그+보너스 EXP | 전권 |
| 호위(NPC) | 제3진영(녹색) 유닛 보호. 사망해도 패배는 아니되 루트 분기 | 아주 호송, 왕처일 후송 |
| 설득 | 특정 아군을 특정 적 인접+선택 → 수회 성공 시 이탈/영입 | 양강, 구천인, 모용가 가신 |
| 출전 제약 | 스테이지별 인원 제한/필수 출전/단독전 | 취현장(소봉+2), 검총(양과 단독) |
| 진영 선택 | 같은 전장을 두 진영 중 하나로 플레이 | 광명정 (육대파 vs 명교) |
| 이탈 조건 | 특정 유닛이 맵 밖 탈출 지점 도달 시 승리 | 만안사 탈출, 왕부 잠입 |

## 4. 캠페인 구성 (전 4권 + 외전)

### 제1권 사조영웅전 (본편 10 + 분기 2)

```
1-1 우가촌의 눈보라 (프롤로그 전투: 곽소천·양철심 조작 — 비극 연출)
1-2 몽골 초원의 소년 (흑풍쌍살)
1-3 장가구의 기연 (황용)
1-4 조왕부의 연회 ──┬─ [선택 A] 1-5a 연경 추격전 — 양강 설득 시도 (설득 2회 성공 → flag: 양강참회)
                    └─ [선택 B] 1-5b 왕부 약고 잠입 — 6인 제한, 왕처일 호위 (성공 → flag: 전진교은혜)
1-6 귀운장의 결전 (구양봉) ※합류
1-7 도화도 구혼 시험 (구양극) — 서브: 주백통으로 사병 3인 격파 → flag: 노완동인정
1-8 강남의 비극 (매초풍 재등장, 참극의 밤 — 패배 이벤트 전투: 5턴 생존)
1-9 [조건 분기] flag 노완동인정 有 → 매초풍의 증언 (talk, 오해 조기 해소)
                無 → 오해의 검 (vs 황약사 — 승리 불가 특수전, 8턴 생존)
1-10 철장봉의 구천인 — 황용 중상 이벤트, 서브: 영고의 오두막 방문 → flag: 영고인연
1-11 일등을 찾아서 (어초경독 4관문 연전 — 소규모 4연속 전투)
1-12 철창묘의 진실 + 화산논검 (광인 구양봉)
  └ 엔딩 분기: 양강참회 有 → 양강 참회 유배 생존 (2권에서 양과 관계 대사 변화)
              無 → 원작대로 철창묘 최후
```

### 제2권 신조협려 (본편 9 + 분기 2)

```
2-1 전진교의 소동 (소년 양과)
2-2 고묘의 사제 (이막수 침입)
2-3 [시점 선택] ┬─ 양과 루트: 2-4a 절정곡 — 공손지, 정화독 (서브: 이막수 격파 → flag: 적련소멸)
               └─ 곽정 루트: 2-4b 영웅대연 — 금륜법왕 첫 대결 (서브: 곽부·곽양 NPC 보호)
2-5 단장애의 이별 (talk, 16년 시간 도약) ※합류
2-6 [외전 해금] 검총의 신조 — 양과 단독 + 신조 NPC, 독고구패 유산 (클리어 → 양과 암연소혼장 강화)
2-7 풍릉나루의 소문 (곽양 시점 talk + 소규모전)
2-8 양양 영웅대회 (곽부의 실수 분기: 서브 목표 실패 시 flag: 곽부반목)
2-9 절정곡 재방문 — 십육 년의 약속 (생존전 → 재회 연출)
2-10 양양 공성전 (금륜법왕 최종, 곽양 구출 호위전)
  └ 엔딩: 화산 3차 논검 + 곽양 여운 (flag: 곽양인연 → 3권 아미 연결)
```

### 제3권 의천도룡기 (본편 9 + 분기 2)

```
3-1 [외전 겸 프롤로그] 소년 장군보 (곽양과 함께 — 2권 flag 곽양인연 시 특별 대사)
3-2 빙화도의 귀환 (장취산 일가 호위전 — 현명이로)
3-3 무당산의 위기 (장무기 한독 이벤트)
3-4 호접곡의 의선 (설득전: 기효부? → 아군 의료 NPC 획득)
3-5 [진영 선택] 광명정 ┬─ 육대파 측 잠입 루트 (성곤 음모 조기 발견)
                      └─ 명교 방어 루트 (6파 파상공세 생존전)
3-6 만안사 탑 (탈출전: 갇힌 고수 3명 이상 구출 시 flag: 육대파신뢰)
3-7 영사도 (금화파파·페르시아 명교 — 해상 맵)
3-8 소림 도사영웅대회 (성곤 최후, 금강복마권진 연전)
3-9 무당 대결전 (현명이로 최종)
  └ 엔딩 분기: 조민 루트 / 주지약 루트 (누적 대화 선택지로 결정, 엔딩 연출 상이)
```

### 제4권 천룡팔부 (본편 10 + 분기 2)

```
4-1 강남의 기연 (단예, 구마지)
4-2 행자림의 진실 (소봉 출생 비밀 — talk + 개방 이탈 전투)
4-3 취현장 혈투 (소봉+2인 한정, 군웅 파상 — 서브: 살상 최소화(적 격파 수 제한) → flag: 인의)
4-4 [호위 분기] 소경호의 갈림길 — 아주 호위전
      성공 → flag: 아주생존 (IF 루트 — 팬메이드 분기임을 명시)
      실패 → 원작 비극 루트 (소봉 침울, 이후 대사 전면 변화)
4-5 진롱기국 (허죽)
4-6 서하 부마 선발 (잠입 + 빙고 결전)
4-7 소실산 대전 1 (모용복·정춘추·구마지)
4-8 소실산 대전 2 (소원산·모용박 진상 → 무명승 이벤트)
4-9 안문관 최후의 결전 (야율홍기 — 방어전)
  └ 엔딩 분기: 아주생존 + 인의 → IF 대단원 (소봉이 웃으며 초원으로)
              그 외 → 원작 여운 엔딩 (이별의 안문관, 순화 연출)
```

### 외전 (해금형 보너스)
| 외전 | 해금 조건 | 내용 |
|---|---|---|
| 월녀검 | 1권에서 한소영 10레벨 | 월녀검법의 기원, 아청 시대 회상전 |
| 검총의 신조 | 2권 진행 중 | 독고구패 유산 (양과 단독전) |
| 활사인묘의 약속 | 2권 클리어 | 왕중양과 임조영 회상전 (고묘 기원) |
| 풍릉야화 | 3권 클리어 + 곽양인연 | 황삼여자의 예고 (신조협려 후일담) |
| 진최종전 영웅집결 | 전 외전 + 주요 선행 플래그 | 시공 초월 총력전 (역대 보스 연합) |

## 5. 엔진 요구사항 (구현 명세)

1. **데이터**: `chapters.json` → `stages.json` (그래프). 필드: `id, book, kind, next, deploy{cap,forced,solo}, objectives[], npc[], persuade[], sides{}`
2. **플래그**: `G.flags` + 대화 스크립트 조건행(`{if}`)/선택지(`{choice}`) + 세이브 v2 (`flags, stageId`), v1 세이브 → 권 시작점 마이그레이션
3. **전투 엔진**: 제3진영(NPC) AI(도주/대기), 설득 커맨드, 탈출 타일, 서브 목표 판정/표시, 출전 제약
4. **UI**: 루트 맵 화면(권별 노드 그래프·클리어/분기 표시), 서브 목표 패널, 플래그 요약(전적부)
5. **호환**: 영웅집결 무한 모드는 현행 유지 (아레나 풀만 신규 맵으로 확대)
