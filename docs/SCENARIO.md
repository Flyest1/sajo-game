# 강호의 별 — 분기형 캠페인 시나리오 설계서 v2.1

> 본 문서는 AI(Claude)가 작성했습니다.
> 김용(金庸) 원작 4부작의 사건·인물 '사실'만을 뼈대로 하며, 게임 내 모든 대사·연출·외전 스토리는 팬메이드 창작입니다.
> 원작·위작의 텍스트 원문은 수집·인용하지 않습니다 (전투 장면은 사건 단위로 목록화하여 재구성).

---

## 1. 캠페인 총구조 (v2.1 확정안)

타이틀에서 캠페인 선택. 역사 시대순 배치.

```
◆ 캠페인 Ⅰ 천룡팔부        (북송 후기 · 독립 캠페인 · 3주인공 루트제)   16~18 스테이지
◆ 캠페인 Ⅱ 사조삼부곡      (남송~원말 · 3권 연속, 플래그 세대 전파)     34~38 스테이지
   제1권 사조영웅전(12~13) → 제2권 신조협려(11~12) → 제3권 의천도룡기(11~13)
◆ 외전 Ⅰ 화산논검 전기     (사조 이전 시대 · 창작 프리퀄)               6 스테이지
◆ 외전 Ⅱ 강호later 후일담  (의천 이후 시대 · 창작 속편)                 5 스테이지
◆ 해금 외전 5종 + 진최종전 영웅집결 (시공 크로스오버는 여기로 이동)
◆ 영웅집결 무한 모드 (유지 · 아레나 확대)
```

- **사조↔신조**: 같은 세이브로 연속 진행. 곽정·황용 성장/플래그 유지, 1권의 선택이 2권 대사·분기에 반영.
- **신조→의천**: 세대 계승 연출(곽양·장군보 브리지 스테이지)로 연결. 로스터는 교체되되 "계승 보너스"(전권 플래그에 따른 초기 무공/아이템) 전달.
- **천룡팔부**: 완전 독립. 크로스오버 없음(본편 순수화). 소봉·단예·허죽의 3루트 → 합류 구조.

## 2. 캠페인 Ⅰ — 천룡팔부 (3주인공 루트제)

### 구조
```
T-0 서장: 안문관의 비극 (30년 전 회상 조작전)
[루트 선택 — 언제든 전환 가능한 3열 병행, 각 루트 클리어 시 다음 열 해금]
소봉편: T-1a 개방 대회(행자림 진실) → T-2a 취현장 혈투(소봉+2 한정, 살상수 제한 서브목표)
        → T-3a 소경호 호위전 (아주 호위: 성공=IF플래그 / 실패=원작 비극)
단예편: T-1b 무량산 검호궁(능파미보) → T-2b 만겁곡(종령 구출) → T-3b 고소 연자오(모용가)
허죽편: T-1c 진롱기국 → T-2c 표묘봉 영취궁(생사부 반란 진압) → T-3c 서하 빙고(동모vs이추수 사이 생존전)
[합류]
T-4 서하 부마 선발 (3주인공 총집결 이벤트+전투)
T-5 소실산 대전 전편 (군웅 vs 3형제 — 모용복·정춘추·구마지)
T-6 소실산 대전 후편 (소원산·모용박 진상 → 무명승 이벤트)
T-7 안문관 최후의 결전 (대요군 방어전 + 야율홍기)
```
### 엔딩 분기 (요청 반영: 다중 엔딩)
| 조건 | 엔딩 |
|---|---|
| 아주생존(IF) + 인의 플래그 | 초원의 약속 — 소봉·아주 해피엔딩 (팬메이드 IF 명시) |
| 아주 사망 + 인의 | 원작 여운 — 이별의 안문관 (순화 연출) |
| 인의 실패(취현장 살상 과다) | 고독한 영웅 — 어두운 여운 엔딩 |
| 단예 호감 누적(왕어언/종령/목완청 선택지) | 단예 개인 에필로그 3종 |
| 허죽 몽고? 서하 인연 플래그 | 허죽 몽고랑(夢姑) 에필로그 |

### 원작 전투 신 커버 목록
무량산 검호궁 / 행자림·개방 대회 / 취현장 혈투 / 소경호 / 진롱기국 / 영취궁 반란 / 서하 빙고 / 서하 부마 선발 / 소실산 전·후편 / 안문관. (누락 없던 취현장·아주 서사·소원산 진상 전부 포함)

## 3. 캠페인 Ⅱ — 사조삼부곡

### 제1권 사조영웅전 (12~13 스테이지)
```
1-1 우가촌의 눈보라 (전사: 곽소천·양철심 조작 비극전)
1-2 대막의 결투 (강남칠괴 vs 흑풍쌍살 — 매초풍 눈 실명 이벤트)
1-3 몽골 초원의 소년 (성장 후 첫 실전)
1-4 장가구의 기연 (황용)
1-5 조왕부의 연회 ─┬ [선택A] 연경 추격전: 목염자 비무초친+양강 설득(2회 성공→flag 양강참회)
                   └ [선택B] 왕부 잠입: 6인 제한, 왕처일 해독 약재 탈취(→flag 전진교은혜)
1-6 귀운장의 결전 (합류 · 구양봉 첫 대면)
1-7 도화도 구혼 삼시험 (구양극 — 서브: 주백통 활약 → flag 노완동인정)
1-8 해상 표류전 (구양봉의 배 — 특수 소맵 연전)
1-9 강남의 참극 (매초풍·양강의 밤 — 5턴 생존 패배이벤트전)
1-10 [분기] 노완동인정 有 → 밀실의 진상 (talk+소전투) / 無 → 오해의 검 (vs 황약사 8턴 생존전)
1-11 철장봉 (구천인 — 황용 중상, 서브: 영고 오두막 → flag 영고인연)
1-12 어초경독 사관문 (일등대사 — 4연속 소전투)
1-13 철창묘의 결말 + 2차 화산논검 (광인 구양봉)
   └ 양강참회 여부로 철창묘 연출·2권 양과 관계 분기
```
전투 신 커버: 우가촌/대막 흑풍쌍살/장가구/조왕부(왕처일 중독)/비무초친/귀운장/도화도 삼시험/해상 표류/우가촌 밀실/철장방/영고/어초경독/철창묘/화산논검. (몽골 서정·사마르칸트 공성은 외전 해금 스테이지로)

### 제2권 신조협려 (11~12)
```
2-1 전진교의 습격 (몽골병 난입 — 양강참회 플래그 시 특별 대사)
2-2 고묘의 사제 (이막수)
2-3 [시점 분기] 양과편: 절정곡(공손지·정화독) ↔ 곽정편: 대승관 영웅대회(금륜 첫 대결)
2-4 만수산장의 계략 (곽부·무씨형제 NPC — 서브 실패 시 flag 곽부반목)
2-5 단장애 (talk · 16년 도약)
2-6 검총의 신조 (양과 단독+신조 NPC — 독고구패 유산, 현철중검 획득)
2-7 풍릉나루 (곽양 시점 도입)
2-8 양양 영웅대회 (곽양 3소원 이벤트)
2-9 절정곡 재방문 (16년의 약속 — 생존전 후 재회)
2-10 양양 공성전 전편 (화공·곽양 납치 호위전)
2-11 양양 대첩 (금륜법왕 최종 + 몽케 격살 이벤트) → 3차 화산논검 (talk 엔딩)
```

### 제3권 의천도룡기 (11~13)
```
3-1 브리지: 소년 장군보 (곽양과 소림 — 2권 곽양 플래그 반영)
3-2 왕반산도 (사손 등장·도룡도 — 조작전)
3-3 빙화도 귀환 호위전 (현명이로)
3-4 무당산 백수 대연 (비극 이벤트+전투)
3-5 호접곡과 서역행 (장무기 성장·아군 재편)
3-6 [진영 선택] 광명정: 육대파 잠입 루트 ↔ 명교 방어 루트 (성곤 음모 발견 시점 상이)
3-7 만안사 탑 탈출전 (구출 수 서브목표 → flag 육대파신뢰)
3-8 무당산 태극 초전수 (조민군 습격 — 장삼봉 NPC)
3-9 영사도 (금화파파·페르시아 삼사 — 해상 맵)
3-10 소림 도사영웅대회 (금강복마권진 연전 + 성곤 최후)
3-11 대도 최종전 (몽골 조정 — 여러 flag 회수)
   └ 엔딩: 조민 루트 / 주지약 루트 (누적 선택지 포인트제) + 육대파신뢰·아주? (독립) 조합으로 에필로그 4종
```

## 4. 외전 캠페인 (창작 · 위작 컨셉 대체)

### 외전 Ⅰ 화산논검 전기 (프리퀄 6 스테이지 — "위작 화산논검"의 컨셉을 자체 창작으로)
젊은 오절 시대. 구음진경을 둘러싼 혈겁 → 왕중양의 결단.
```
G-1 황실 서고의 기서 (구음진경 유래 — 잠입전)  G-2 강호 혈겁 (경전 쟁탈 난전)
G-3 임조영과 왕중양 (고묘 — 협력전+감정 서사)  G-4 젊은 사절의 도전 (동사·서독·남제·북개 연전)
G-5 전진 vs 백타산 (구양봉 야습)               G-6 1차 화산논검 (7일 논검 — 보스 러시)
```
클리어 보상: 본편에서 왕중양류 무공 계승 요소 해금.

### 외전 Ⅱ 강호 후일담 (속편 5 스테이지 — "의천 외전" 컨셉을 자체 창작으로)
명 건국 직후. 은퇴한 장무기·조민, 아미·무당 차세대.
```
E-1 몽골 초원의 재회 (장무기·조민 은둔처 습격)  E-2 도룡도의 행방 (신흥 세력)
E-3 아미의 후계 (주지약 이후)                    E-4 무당 2세대 (송청서의 그림자)
E-5 최후의 협 (신 무림 연합전 — 다중 엔딩: 3권 엔딩 플래그 반영)
```

### 해금 외전 (기존안 유지)
월녀검 / 검총의 신조(2-6로 본편 승격됨→대체: 독고구패 회상 5단계 검도전) / 왕중양·임조영 활사인묘(외전Ⅰ와 연동) / 풍릉야화·황삼여자 / **진최종전 영웅집결** (전 캠페인 클리어 — 시공 크로스오버 부활: 소봉×곽정×양과×장무기 총집결 vs 역대 보스 연합)

## 5. 분기 시스템·엔진 요구사항 (v2 유지 + 추가)

기존 설계(§3 스테이지 그래프·플래그·서브목표·호위 NPC·설득·출전 제약·진영 선택·탈출전)에 추가:

1. **캠페인 선택 화면** + 캠페인별 독립 세이브 슬롯 (flags, stageId, roster)
2. **병행 루트 보드** (천룡 3열 루트 UI — 열 전환/해금 표시)
3. **호감도 포인트** (대화 선택지 누적 → 조민/주지약, 단예 에필로그 판정)
4. **계승 보너스** (권 경계에서 플래그 → 차권 초기 혜택 변환 테이블)
5. **패배 이벤트전** (패배해도 진행되는 전투 — 승리 시 보너스 플래그)
6. 세이브 v2 스키마 + 기존 v1 세이브 → "클래식 완주 기록"으로 보존 (무한 모드 기록 유지)
