# 강호의 별 — 밸런스 패스 리포트 (R12)

> 본 문서는 AI(Claude)가 작성했습니다.
> 기준일: 2026-07-18 · 도구: `npm run balance` (tools/balance_sim.mjs)

## 방법

전투 엔진의 `calcStrike` 공식(상성·기술 배수·명중·필살·2연격)을 오프라인으로 재현하고,
캐릭터 성장 데이터(`characters.json`의 grow)로 기대 성장치를 계산해
**대표 매치업**의 난이도별 지표를 산출한다. DOM/브라우저 없이 순수 수식으로 검증한다.

- 아군: 캠페인 표준 레벨대의 기대 성장치(장비·협공·인연·숙련 **미적용** = 하한 조건)
- 적: 고정 base 스탯 × 난이도 배율(story 0.85 / std 1.0 / hero 1.15) — 엔진과 동일
- 지표: 명중률(%), 처치 소요 라운드(기대 피해·명중·필살·2연격 반영)
- 이상치 자동 플래그: 즉사위험(적→아군 <1.2R), 적 즉사(<1.0R), 명중 과소(<45%), 교착(양측 >12R)

## 결과 요약

**표준(std) 난이도의 곡선은 건전하다.** 대표 아군은 잡졸을 1.5~2.5R, 중간 보스를 3~6R에
처치하며 잡졸에게는 거의 피해를 입지 않는다(방어전·다수전에서 자연스러운 긴장 유지).

최상위 보스(구양봉·금륜법왕·야율홍기 등)는 아군 단독·무장비 기준으로 처치에 오래 걸린다.
이는 **의도된 설계**로, 실제 플레이에서는 해당 보스를 (1) 더 높은 레벨, (2) 병기/보구 장착,
(3) 협공·인연 보정, (4) 다수 아군, (5) 무공 숙련으로 상대하도록 배치되어 있다.
특히 진최종전은 시작부터 Lv12·전설 장비·12영웅 편성을 제공한다.

## 이상치와 판단

| 플래그 | 판단 |
|---|---|
| hero 구양봉 → 곽정 1.2R (즉사위험) | 서독은 사조 종반의 최상위 보스이며 단독 상대 대상이 아님. hero는 명시적 도전 난이도("도전적인 강호"). 상수 조정 없이 유지 — 표준 곡선을 해치지 않기 위함 |
| 최상위 보스 처치 장기화(무장비 기준) | 장비·협공·인연·숙련·레벨로 해소되는 정상 범위. 상수 불변 |

**결론:** 표준 난이도 곡선이 목표 범위 안에 있어 전역 상수(명중 기저·상성 가중·난이도 배율)
조정은 하지 않는다. 최상위 보스는 팀 편성·장비 전제의 도전 요소로 유지한다. 향후 콘텐츠 추가 시
`npm run balance`로 회귀 점검한다.
