## 로드맵 (우선순위 순)

### R7 — 지원 대화·인연 시스템 ✅ 완료 (2026-07-18)
- **인연 페어 데이터**: 20쌍(곽정·황용, 양과·소용녀, 장무기·조민/주지약, 소봉·아주 등) — `src/data/supports.json`, 랭크별 C/B/A 창작 대사
- **전장 인연 보정**: 인연 페어가 인접하면 피해 +랭크·명중 +랭크×4·필살 +랭크×2, 수비 측 회피 +랭크×3 (calcStrike 반영, 전투 예측에 인연 배지 표시)
- **거점 지원 대화**: 거점 '지원 대화' 탭 → 대화 시청 시 랭크 1단계 상승(거점당 페어별 1회 제한), 세이브에 랭크 저장
- **합격 각성**: ★표시 5쌍(곽정·황용, 양과·소용녀, 소봉·아주 등) A랭크 도달 시 특별 연출 문구

### R8 — 난이도·플레이 옵션 (시스템)
- **난이도 3종**: 이야기(적 능력 -15%·경험치 +30%) / 표준(현행) / 협객(적 +15%·증원 강화·자금 -30%)
  - 캠페인 시작 시 선택, 세이브에 기록, 중간 하향만 허용(FE 관례)
- **전투 속도 옵션**: 연출 1x/1.5x/2x + 적 페이즈 스킵(결과만 표시)
- **설정 모달 통합**: 사운드·BGM 볼륨 분리 슬라이더·연출 속도·난이도 표시 (설정 저장)

### R9 — 해금 외전 추가분 (콘텐츠)
- **월녀검 전설** (3스테이지): 아청의 검 — 고대 시대 프리퀄 소품
- **독고구패 회상 검도전** (5연전): 이기어검까지 5단계 검의 경지 — 양과 검총 클리어 시 해금
- **활사인묘 비사** (3스테이지): 왕중양·임조영 — 외전Ⅰ 클리어 연동
- **풍릉야화·황삼여자** (3스테이지): 신조 이후 고묘파 — 의천 클리어 연동
- 해금 조건 표시 UI(잠금 카드에 조건 명시), 클리어 보상은 진최종전용 강화 아이템

### R10 — 업적·기록·회차 (리텐션)
- **업적 배지** 30종(무피해 클리어·전원 생존·N턴 격파·전 보물 회수·전 엔딩 수집 등) + 업적 화면
- **전적 통계**: 캠페인별 클리어 턴수·격파수·MVP 집계
- **회차 플레이(주회)**: 완주 캠페인 2회차 시작 시 계승 선택(자금/무공/랭크 중 택1) + 회차 전용 칭호

### R11 — PWA·조작성 (플랫폼)
- **PWA**: manifest + service worker 오프라인 캐시 → 홈 화면 설치, 오프라인 플레이
- **키보드 단축키**: 방향키 커서·Enter 선택·Esc 취소·E 턴종료 / **게임패드** 기본 대응
- **세이브 백업**: 전 기록 JSON 내보내기/가져오기(기기 간 이동)

### R12 — 연출 고도화·밸런스 패스 (마감 품질)
- **이벤트 컷신**: 주요 장면(양양 함락·안문관 등) 일러스트풍 SVG 컷 + 카메라 패닝
- **전장 날씨/시간**: 눈·비·야전 오버레이(외전Ⅰ 설산전 등 무드 연출, 일부 지형 효과 연동)
- **밸런스 패스**: 전 캠페인 자동 시뮬레이션(AI vs AI 통계)으로 난이도 곡선 조정
- **BGM 확장**: 캠페인별 테마 스케일·템포 변형(천룡=비장, 외전=고풍 등)

## 작업 방식 (기존 관례 유지)

각 단계: 데이터/엔진 구현 → `npm run validate` → Playwright 스모크(신규 기능 경로 + 회귀) → 스크린샷 확인 → 커밋/푸시 → GitHub Pages 배포 확인. 콘텐츠 대사는 전부 원작 사실 기반 팬메이드 창작(원문 인용 없음).
