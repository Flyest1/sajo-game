## 로드맵 (우선순위 순)

### R7 — 지원 대화·인연 시스템 ✅ 완료 (2026-07-18)
- **인연 페어 데이터**: 20쌍(곽정·황용, 양과·소용녀, 장무기·조민/주지약, 소봉·아주 등) — `src/data/supports.json`, 랭크별 C/B/A 창작 대사
- **전장 인연 보정**: 인연 페어가 인접하면 피해 +랭크·명중 +랭크×4·필살 +랭크×2, 수비 측 회피 +랭크×3 (calcStrike 반영, 전투 예측에 인연 배지 표시)
- **거점 지원 대화**: 거점 '지원 대화' 탭 → 대화 시청 시 랭크 1단계 상승(거점당 페어별 1회 제한), 세이브에 랭크 저장
- **합격 각성**: ★표시 5쌍(곽정·황용, 양과·소용녀, 소봉·아주 등) A랭크 도달 시 특별 연출 문구

### R8 — 난이도·플레이 옵션 ✅ 완료 (2026-07-18)
- **난이도 3종**: 이야기(적 -15%·경험치 +30%·자금 +20%) / 표준 / 협객(적 +15%·경험치 -10%·자금 -30%) — 새 시작 시 캡처, 세이브에 기록
- **전투 속도 옵션**: 연출 ×1/×1.5/×2 + 적 페이즈 빠르게(가속) — aSleep 전역 배속
- **설정 모달 통합**: 난이도·연출 속도·적 페이즈·사운드 (타이틀·전투 ⚙ 버튼, 설정 저장)
- **무공 숙련도**(추가): 스킬 사용 횟수 누적 → 5단계(初~極) 숙련, 위력·명중 +·기 소모 - (전 모드 공유 저장)
- **아이템 확장**(추가): 병기·보구·영약 11종 신규(유엽도·판관필·설룡검·연위갑·금사연갑·칠성패·비호혜·소환단·벽하환·정심단 등), 거점 상점에 기본 카탈로그 항상 병합 노출
- **UX**(추가): 화면 배율 ×1.25 추가, 적 턴 카메라 자동 추적, 모바일 세로 시 정보/상태 팝업을 맵 아래 배치

### R9 — 해금 외전 추가분 ✅ 완료 (2026-07-18)
- **월녀검 전설** (3전투): 한소영의 월녀검법 전승 — 사조영웅전 완주 시 해금
- **독고구패 검도전** (5연전): 이검→연검→중검→목검→무검 5경지, 양과 단독(Lv11·현철중검 시작 부스트) — 신조협려 완주 시 해금
- **활사인묘 비사** (3전투): 왕중양·임조영, 화산 세이브 계승 — 외전Ⅰ 완주 시 해금
- **풍릉야화** (3전투): 소용녀·곽양 고묘파, 황삼여자의 유래 — 의천도룡기 완주 시 해금
- 잠금 카드에 개별 해금 조건·진행도(0/1) 표시(범용 lockedCard), 캠페인 선택 화면 '해금 외전' 구획

### R10 — 업적·기록·회차 ✅ 완료 (2026-07-18)
- **업적 배지 24종**: 첫 승리·무피해·3턴 클리어·전 보물·누적 격파/보스/필살·숙련 極·인연 A·승급·무한 10/20파·캠페인별 완주·아주 IF·전 캠페인 완주 등 + 달성 토스트
- **전적 통계**: 승리·격파·보스·필살 누적, 무한 최고파, 완주 캠페인 목록 — 타이틀 '기록·업적' 화면
- **회차(New Game+)**: 완주 캠페인 카드에 '회차+' — 계승 보너스 택1(자금 +2000 / 명품 병기 / 인연 전체 B랭크 시작)
- 숙련도는 이미 전 모드 공유 저장이라 회차에도 자동 계승

### R11 — PWA·조작성 (플랫폼)
- **PWA**: manifest + service worker 오프라인 캐시 → 홈 화면 설치, 오프라인 플레이
- **키보드 단축키**: 방향키 커서·Enter 선택·Esc 취소·E 턴종료 / **게임패드** 기본 대응
- **세이브 백업**: 전 기록 JSON 내보내기/가져오기(기기 간 이동)

### R12 — 연출 고도화·밸런스 패스 (마감 품질)
- **이벤트 컷신**: 주요 장면(양양 함락·안문관 등) 일러스트풍 SVG 컷 + 카메라 패닝
- **전장 날씨/시간**: 눈·비·야전 오버레이(외전Ⅰ 설산전 등 무드 연출, 일부 지형 효과 연동)
- **밸런스 패스**: 전 캠페인 자동 시뮬레이션(AI vs AI 통계)으로 난이도 곡선 조정
- **BGM 확장**: 캠페인별 테마 스케일·템포 변형(천룡=비장, 외전=고풍 등)

## 작업 방식 (기존 관례 유지)

각 단계: 데이터/엔진 구현 → `npm run validate` → Playwright 스모크(신규 기능 경로 + 회귀) → 스크린샷 확인 → 커밋/푸시 → GitHub Pages 배포 확인. 콘텐츠 대사는 전부 원작 사실 기반 팬메이드 창작(원문 인용 없음).
