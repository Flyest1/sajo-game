# 강호의 별 — 게임 시스템 설계서 (R1)

> 본 문서는 AI(Claude)가 작성했습니다.
> FE·조조전류 SRPG의 일반적 시스템 관습(장비/상점/거점/승급/보물)을 참고하되, 데이터·구현은 전부 자체 설계입니다.

## 1. 재화·아이템

- **금전(兩)**: 전투 승리 보수 + 보물 획득. 캠페인별 독립.
- **아이템 분류**
  | 종류 | 슬롯 | 예 | 효과 |
  |---|---|---|---|
  | 병기(weapon) | 유닛당 1 | 목검→철검→강철검→보검 | 공격+, 명중+, 필살+ |
  | 보구(acc) | 유닛당 1 | 호신부·옥패·경공혜·웅담환 | 방어/정신/이동/HP+ |
  | 영약(use) | 공용 행낭 | 금창약·대환단·해독환 | 전투 중 "도구" 커맨드로 사용 (행동 소모) |
  | 비급(key) | 공용 행낭 | 현문비급 등 | 승급 재료 |
- 행낭(인벤토리)은 부대 공용(조조전식). 장비는 유닛별 장착.

## 2. 거점(트영지) — 스테이지 사이 인터미션

캠페인 그래프의 `camp` 노드 또는 출전 화면에서 진입.
- **편성**: 장비 장착/해제, 승급 실행, 부대 현황
- **상점**: 구입 / 매각(반값). 스테이지 진행에 따라 품목 확장
- **행낭**: 보유 아이템 확인
- **출발**: 다음 스테이지로

## 3. 승급(클래스 진화)

- 조건: 레벨 N 이상 + 지정 비급 소지 → 거점에서 실행
- 효과: 클래스명 변화 + 능력치 보너스 + 신규 무공 습득 (예: 왕중양 → 中神通, 일양지 개안)
- 캐릭터 데이터 `promo:{lvl, item, cls, bonus, skill}` 로 정의. 주요 아군 전원에 단계적 적용 예정.

## 4. 전장 보물

- 스테이지 데이터 `treasures:[{x,y,item|gold}]` — 맵에 궤짝 표시
- 아군 유닛이 해당 칸에서 행동을 마치면 획득 (적은 획득 불가)
- 파밍 동선 vs 진격 속도의 트레이드오프 설계용

## 5. 분기 캠페인 엔진 (v2)

- **캠페인 선택**: 클래식(기존 19장)·신규 캠페인·무한 모드 병행. 캠페인별 독립 세이브(`kimyong_v2_<id>`)
- **스테이지 그래프**: 노드 kind = `battle`(전투) / `camp`(거점) / `choice`(루트 선택) / `end`(엔딩)
  - `next`: 문자열(단일) 또는 `{prompt, options:[{label, to, set}]}`(선택 분기)
- **플래그**: 선택지·전투 결과가 `flags`에 기록 → 이후 대사 조건행(`{if}`)·분기·엔딩 결정
- **전투 확장**: 출전 제약(필수/인원), 승리 보수(금전), 보물, 도구 커맨드
- **루트 맵**: 진행 그래프 시각화(클리어/현재/잠김/분기 갈래)
