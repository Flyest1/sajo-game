**▶ 플레이: https://flyest1.github.io/sajo-game/**

**v2 리부트 로드맵 R1~R6 완결** — 분기형 캠페인 7종 + 클래식 19장 + 영웅집결 무한 모드.

시스템: 공용 행낭·장비(무기/보구)·상점·거점 인터미션 · 전장 보물 · 클래스 승급 ·
호감도/플래그 분기 · 출전 제약(필수/인원/루트 한정) · 계승 보너스 · 루트 맵.

연출·사운드(2단계): 유닛 이동 트윈 · 공격 런지/피격 섬광/필살 셰이크 · 화면 전환 페이드 ·
WebAudio 절차 합성 효과음 + 오음계 BGM(외부 오디오 자산 없음) · 사운드 토글(설정 저장).

그래픽(3단계): 좌표 시드 변형 지형 타일(나무 2형·풀·들꽃·자갈·이끼·설봉·기와 가옥·등롱) ·
유닛 토큰 링 그라디언트/보스 오라/선택 회전 링 · 대화 배경 무드 4종(새벽/낮/황혼/밤) ·
타이틀 달무리·별·안개 드리프트 · 결과 화면 낙관(도장) 연출. 전부 코드 생성 SVG(외부 이미지 없음).

UX(4단계): 전체 화면 전투(맵 자동 확대 채움) · 고정 상단바 · 캐릭터 클릭 시 팝업 상태 카드 ·
정보/기록 버튼 토글 · 미니맵(뷰포트 표시·클릭 이동) · 이어하기 통합 세이브 허브(클래식+캠페인+무한).
향후 계획: docs/PLAN.md (R7 지원 대화 ~ R12 밸런스 패스).

## 개발

```bash
npm install
npm run dev        # 로컬 개발 서버 (모바일은 같은 네트워크에서 접속 가능)
npm run validate   # 게임 데이터 무결성 검증
npm run build      # dist/ 프로덕션 빌드
npm run preview    # 빌드 결과 미리보기
```

## 구조

```
index.html               Vite 엔트리
src/
  main.js                부팅 · 전역 노출
  style.css              전체 스타일 (반응형/모바일 포함)
  data.js                데이터 로더 · 상성 규칙
  data/*.json            챕터·캐릭터·무공·지형·엔딩 데이터 (콘텐츠는 여기만 수정)
  gfx.js                 SVG 그래픽 (초상화 생성기·타일·유닛 토큰)
  sfx.js                 사운드 (WebAudio 절차 합성 효과음·BGM)
  game.js                전투 엔진 · 화면 흐름 (이동/전투/AI/무한 모드)
tools/validate_data.mjs  데이터 검증 (CI에서 빌드 전 실행)
legacy/                  구버전 단일 파일 백업
```

## 배포

`main` 브랜치에 push → GitHub Actions가 검증·빌드 후 GitHub Pages로 자동 배포.
(저장소 Settings → Pages → Source 를 **GitHub Actions** 로 설정 필요)
